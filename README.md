## Stack
- React
- TypeScript
- Vite
- shadcn/ui

- Node.js (runtime)

## Prasyarat
- Node.js (LTS direkomendasikan)
- npm atau pnpm/yarn

## Instalasi & Menjalankan (singkat)
1. Clone repository:
```bash
git clone https://github.com/AzrilFitrayana/-SIMS-PPOB-Muhammad-Azril-Fitrayana.git
cd -SIMS-PPOB-Muhammad-Azril-Fitrayana
```

2. Pasang dependensi:
```bash
npm install
# atau
# pnpm install
# yarn
```

3. Jalankan server development:
```bash
npm run dev
```
Buka alamat yang ditampilkan (mis. http://localhost:5173).

4. Build untuk produksi:
```bash
npm run build
```

5. Preview hasil build:
```bash
npm run preview
```

Catatan: Jika menggunakan environment variables, buat file `.env` di root dan tambahkan variabel yang diperlukan (mis. `VITE_API_URL`).
